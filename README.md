# FitTrack - Mini Projeto Fullstack

Esse projeto visa ser um app de registro de treinos, com a possibilidade de criar exercícios, divisões/dias e sessões de treino para cada usuário, autenticado via Token JWT.

### Tecnologias utilizadas:
- Front: React + Vite + TS
- API: Node + Express + TS 
- DB: MongoDB Atlas & PostgreSQL (cada um com sua URL própria)

### Como clonar:
```bash
git clone https://github.com/tuca-janahu/mini-projeto-front.git fittrack
cd fittrack/frontend
```
### Instalar dependências e rodar o projeto
```bash
npm i
npm run dev
```
